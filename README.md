# Schwung MVerb

Audio FX module for [Schwung](https://github.com/charlesvestal/schwung) on Ableton Move.

Algorithmic reverb wrapping Martin Eastwood's MVerb engine with 9 parameters for rich, detailed reverb tails.

**Author:** Andre Louis

## Controls

- **Knob 1:** Damping
- **Knob 2:** Density
- **Knob 3:** Bandwidth
- **Knob 4:** Decay
- **Knob 5:** Predelay
- **Knob 6:** Size
- **Knob 7:** Gain
- **Knob 8:** Mix
- **Menu:** Early Mix

## Build

```bash
./scripts/build.sh
```

## Install

```bash
./scripts/install.sh
```
